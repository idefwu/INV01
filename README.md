# 庫存管理系統

基於JavaScript開發的完整商品進銷存管理系統，提供直觀的Web界面和完整的庫存管理功能。

## 功能特色

### 🏪 基本資料管理
- **商品主檔管理**
  - 商品編號管理
  - 品名、規格、單位設定
  - 售價與成本價維護
  - 安全庫存設定

- **客戶主檔管理** (可選)
  - 客戶基本資料
  - 聯絡方式管理

- **供應商主檔管理** (可選)
  - 供應商基本資料
  - 聯絡方式管理

### 📦 進貨管理
- 建立進貨單（供應商、商品、數量、單價）
- 收貨作業自動更新庫存數量與成本
- 進貨記錄查詢與檢視

### 🛒 銷貨管理
- 建立銷貨單（客戶、商品、數量、單價）
- 自動扣減庫存（庫存不足時阻止銷售）
- 銷貨記錄查詢與檢視

### 📊 庫存管理
- 商品即時庫存數量查詢
- 低庫存自動警示（基於安全庫存設定）
- 手動庫存調整（增加/減少/設定）
- 庫存調整記錄追蹤

### 📈 報表查詢
- **庫存報表**：商品即時數量、成本價值統計
- **進銷存明細表**：指定日期區間內的進貨、銷貨、結存數量與金額分析

## 系統架構

### 核心檔案
```
/
├── inventory-system.js    # 核心業務邏輯
├── index.html            # Web用戶界面
├── 庫存管理規格.md        # 系統需求規格
└── README.md             # 說明文件
```

### 主要類別
- `InventorySystem` - 系統核心控制器
- `Product` - 商品資料模型
- `Customer` - 客戶資料模型
- `Supplier` - 供應商資料模型
- `PurchaseOrder` - 進貨單資料模型
- `SalesOrder` - 銷貨單資料模型
- `InventoryAdjustment` - 庫存調整記錄模型

## 快速開始

### 系統需求
- 現代化網頁瀏覽器（Chrome、Firefox、Safari、Edge）
- 無需伺服器環境，可直接在瀏覽器中運行

### 安裝與使用
1. 下載所有檔案到本地資料夾
2. 用瀏覽器開啟 `index.html`
3. 開始使用系統功能

### 基本操作流程
1. **初始設定**
   - 新增供應商資料
   - 新增客戶資料（可選）
   - 建立商品主檔

2. **進貨作業**
   - 建立進貨單
   - 執行收貨作業更新庫存

3. **銷貨作業**
   - 建立銷貨單
   - 系統自動扣減庫存

4. **庫存管理**
   - 監控低庫存警示
   - 執行庫存調整作業

5. **報表分析**
   - 查看庫存報表
   - 分析進銷存明細
